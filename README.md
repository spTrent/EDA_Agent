# Data Science AI Agent. Track D

Локальный ИИ-агент для автоматизации первичного анализа данных и построения бейзлайн-моделей. Работает на базе локальной LLM через Ollama

## Возможности агента

Пайплайн агента включает три основных инструмента (tools):
* **analyze_dataset**: Парсинг и анализ CSV-файлов. Автоматически выявляет константные признаки, избыточные пропуски (более 80%) и возможные утечки данных (Data Leakage) на основе корреляционного анализа.
* **train_baseline**: Обучение регрессионной модели градиентного бустинга (CatBoost). Включает автоматический выбор стратегии валидации (K-Fold CV для малых выборок или Train/Test split), расчет метрик (R2, MAE) и извлечение важности признаков (Feature Importance).
* **save_report**: Генерация и сохранение итогового аналитического отчета в формате Markdown.

## Технологический стек

* Язык: Python 3
* LLM: Qwen 2.5 7B (локально через Ollama)
* ML/EDA: `pandas`, `numpy`, `scikit-learn`, `catboost`
* Интеграция: `ollama-python`

## Установка и запуск

1. Установите Ollama и скачайте веса модели:
   ```bash
   ollama pull qwen2.5:7b
   ```

2. Установите необходимые Python-зависимости:
   ```bash
   pip install pandas numpy scikit-learn catboost ollama
   ```

3. Поместите ваш целевой датасет (например, `prices.csv`) в корневую директорию проекта и задайте промпт в app.py.

4. Запустите агента:
   ```bash
   python app.py
   ```
